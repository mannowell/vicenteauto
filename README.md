## 🚀 Deploy

### Vercel

1. Fazer push do repositório para o GitHub
2. Ir a [vercel.com](https://vercel.com) e importar o repositório
3. O deploy é automático — site estático sem build necessário

### Netlify

1. Fazer push do repositório para o GitHub
2. Ir a [netlify.com](https://app.netlify.com) e importar o repositório
3. Publish directory: `.` (raiz do projeto)
4. Não é necessário build command

### GitHub Pages

1. Ir a Settings > Pages no repositório
2. Selecionar a branch `main` e pasta raiz
3. O site estará disponível em `username.github.io/nome-do-repo`
